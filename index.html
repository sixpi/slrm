    slrm (simple local repository manager)

    Helps me keep track of which repositories I have checked out locally have changes which I haven't committed. Future planned features include not just showing how many uncommitted changes there are, but how many commits ahead/behind the local copies are from the remotes ones (especially for git). Nice features like autocomplete, better install process, etc. may also come at some point...

    Usage

    slrm --help
